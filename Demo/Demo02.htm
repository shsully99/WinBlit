Once you have installed WinBlit you will see a new button on the toolbars which appear at the top of Microsoft Word, Internet Explorer and the Outlook e-mail window. The button will contain an icon like this .

Internet Explorer Toolbar containing WinBlit button (on the right hand side of screen)

Microsoft Word Toolbar containing WinBlit button (on bottom left of screen)

Microsoft Outlook Toolbar containing WinBlit button (on the right hand side of screen). This only appears on the toolbar of open e-mails in Outlook.

Whenever you click the button in any of these programs (or its associated menu entry), the WinBlit Window will appear to allow you to speed read the contents of the word document, email or web page.

WinBlit also appears in a standalone application called BlitPad which you can start from the Windows start menu. BlitPad allows you to speedread any text file as well as the contents of the Windows clipboard.

As previously mentioned in this demonstration WinBlit is shown inside a web page. If you are using WinBlit from Outlook, Word or BlitPad it appears inside its own dialog box. In Internet Explorer it will appear in its own band at the bottom of the browser.
