Word

When you open Word you will see the WinBlit logo on the toolbar. Click this button to start WinBlit.

Alternatively - select ‘WinBlit’ from the Tools menu. You can then read the text of the currently open document from within the WinBlit Window.

NOTE:If you would like to read just a portion of the text of the word document then you can select it with the mouse (as you usually do when selecting text in a word processor). Start up WinBlit (via the toolbar or the menu) to just bring the selected text into the WinBlit window.

Note – you cannot currently resize the WinBlit screen once it is open. It automatically assumes the size of the Window of Word. To resize it you must therefore close the WinBlit Window, resize the Word Window and start WinBlit again.
