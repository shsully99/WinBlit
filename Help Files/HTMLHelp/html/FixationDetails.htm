Fixation Details Tab

This tab allows you to change the height and width of fixation for the current configuration.

Fixation Height

Change the value of the fixation height. The fixation height is the number of rows of text highlighted in the current fixation.

Fixation Width

Change the value of the fixation width. The meaning of fixation width is dependent on the Configuration type.

Leave Fixations Highlighted

If this box is checked then a fixation will be left highlighted in the WinBlit Window when the cycle period for the current fixation is finished.

Click sound per fixation

Check this box to produce a click sound at each fixation. Some users have said that this aids their comprehension.
