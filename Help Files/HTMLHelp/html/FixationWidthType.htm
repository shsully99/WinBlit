Fixation Width Type

Defines how the width of a fixation is calculated. It has 3 possible values Words, Characters and Columns.

Words

The width of a fixation is based on the number of words within a fixation. Change the Fixation Width field on the Configuration Tab to set the maximum number of words within a fixation.

Characters

The width of a fixation is based on the number of characters within a fixation. Change the Fixation Width field to set the maximum number of characters within a fixation. Spaces are counted as characters when the width of a fixation is calculated.

Columns

The line is divided into a number of equal width columns - each column represents a fixation. The number of columns and hence fixations is determined by the value you enter in the Fixation Width field of the Configuration Tab. The system puts a word into a fixation depending upon which column the text of the word falls into.
