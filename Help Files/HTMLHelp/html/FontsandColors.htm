Fonts and Colours Tab

This tab allows you to change the fonts and colours of the Text Window.

Back Colour

Click this button to select the WinBlit Window background colour from the colour selection dialog.

Screen Fill Colour

Click this button to select the colour immediately surrounding the text on the screen from the colour selection dialog.

Screen Text Colour

Click this button to change the colour of the text displayed on the screen,

Font

Click this button to change the text font from the Font Selection dialog box.

Fixation Fill Colour

Click this button to select the colour immediately surrounding the text in a fixation.

Fixation Text Colour

Click this button to change the colour of the text when it is displayed within a fixation.
