General Tab

Show Toolbar

Check this to show the Toolbar in the speedreading window.

Show Trackbar

Check this to show the Trackbar in the speedreading window.

Show Statusbar

Check this to show the Statusbar in the speedreading window.

Show Metafile,

Check this to show the Metafile in the speedreading window.

AutoStart BlitPad

If this is checked then BlitPad will automatically start when a document is opened.

AutoStart Internet Explorer

If this is checked then WinBlit will automatically start speedreading when opened in Internet Explorer.

AutoStart Outlook

If this is checked then WinBlit will automatically start speedreading when opened in Microsoft Outlook.

AutoStart Word

If this is checked then WinBlit will automatically start speedreading when opened in Microsoft Word.

Clear Statistics

Click this button to zero the statistics that appear on the WinBlit status bar. This function is also available from the context menu.
