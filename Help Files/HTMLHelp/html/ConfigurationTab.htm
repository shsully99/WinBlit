Configuration Tab

This tab allows you to change the current configuration of WinBlit, add a new one or delete an existing one.

Configuration Name

Select the name of the Configuration you want to change to. When WinBlit is installed it contains one sample Configuration for every possible Configuration Type.

Add Configuration Button

Click this button to add a new configuration to WinBlit.

Remove Configuration Button

Click this button to delete the currently selected Configuration. This will completely remove this Configuration from the system.
