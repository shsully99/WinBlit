Registration

If you have a beta or trial version of the product you are limited in the functionality you can use and the time period of use. You must pay for a license key to activate full functionality in the software.

Go to www.winblit.com on the World Wide Web and select the Purchase link. Enter your credit card details and e-mail address. You will then receive a license key via e-mail within 24 hours.

To activate WinBlit click the Start button then Select Programs. Select WinBlit then BlitPad.

From the BlitPad window : -

EITHER

Select the Help Menu option then select Register.

OR

Click the red ‘R’ button that you can see on the right hand side of the toolbar menu.

You will see a dialog box with 2 boxes ‘Name’ and ‘Registration Key’.

Enter the Name and Registration ID exactly as they were in the e-mail you have received containing the registration key and click the OK button.

If you have entered those details correctly you will now have a fully licensed and therefore fully functional version of the software.

(The red ‘R’ will no longer appear in the WinBlit Window and the Registration menu option will appear grayed out.)
