Statusbar

The Statusbar is shown at the bottom of the window. It has 4 different panes.

  • The left-hand pane displays a text string that describes the current status of reading such as ‘Paused’ or ‘Displaying’.

  • The second pane displays the average words per minute being read using the current reading speed since the application was started.

  • The third pane displays the number of words read since the application was started.

  • The fourth pane the number of seconds elapsed since reading began.
