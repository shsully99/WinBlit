Outlook

When you open an email in Outlook you will see the WinBlit logo on the toolbar. Click this button to start WinBlit.

You can also do this by selecting ‘WinBlit’ from the Tools menu. You can then read the text of the currently open e-mail from within the WinBlit Window.

Note - you cannot currently resize the WinBlit Window once it is open. It automatically assumes the size of the currently open e-mail. To resize it you must therefore close the WinBlit Window, resize the e-mail window and start WinBlit again,
