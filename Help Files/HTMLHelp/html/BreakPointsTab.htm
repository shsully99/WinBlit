Breakpoints Tab

This tab allows you to view, add and remove Breakpoints.

Add Breakpoint

Click this button to add a new Breakpoint to the Current Configuration.

Remove Breakpoint

Click this button to remove an existing Breakpoint from the system. You must firstly click on the name of the Breakpoint you wish to remove.

Modify Breakpoint

Click this button to modify an existing Breakpoint . You must firstly click on the name of the Breakpoint you wish to change.
