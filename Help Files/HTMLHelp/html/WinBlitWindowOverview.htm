WinBlit Window Overview

The WinBlit Window is divided into the Toolbar, the Text Window the Trackbar and the Statusbar.

It always appears the same whichever of the programs you are using it from i.e. Word, Internet Explorer, Outlook or BlitPad.

You can control what it does and how it does it by either clicking toolbar buttons, pressing shortcut keys or right-mouse clicking on the context menu and selecting a function.
