Shortcut keys

To use the shortcut keys you must firstly switch focus to the WinBlit Window. Clicking the mouse in the centre of the WinBlit Screen will switch focus to the WinBlit Window.

If you are not using a keyboard to operate WinBlit please refer to the Notes on Focus.

Start/Stop

Depress ‘S’ to start reading. If currently reading depress this key to stop. You can also do this by depressing the spacebar.

Pause

Depress ‘P’or depress to toggle whether reading is paused or not. You can also do this by depressing <CTRL> (i.e. the control key) and the spacebar at the same time. 

Settings

Depress ‘E’ to open Configuration dialog box.

Page Down

Depress Page Down key to scroll down a page of text in the Text Window.

Page Up

Depress Page Up key to scroll up a page of text in the Text Window.

Line Up

Depress Up Arrow to move up 1 line of text in the Text Window.

Line Down

Depress Down Arrow to move down 1 line of text in the Text Window.

Top of Data

Depress <CTRL> <HOME> to move the Text Window to the top of the data.

Bottom of Data

Depress <CTRL> <END> to move the Text Window to the bottom of the data.

Increase Speed

Depress <CTRL> and Up Arrow to increase the speed of reading. You will see the number of fixations per minute changing in the second panel of the status bar at the bottom of the screen.

Decrease Speed

Depress <CTRL> and Down Arrow to decrease the speed of reading. You will see the number of fixations per minute changing in the second panel of the status bar at the bottom of the screen.

Notes on Focus :-

Although it is possible to operate multiple programs simultaneously inWindows only one program at a time in Windows can be "in focus" and able to accept keyboard input. To use shortcut keys inWinBlit not only must the client program be "in focus" but the Text window itself. The following points should help you in moving focus to the Text window.

  • For BlitPad focus will automatically be moved to the Text Window once BlitPad is started.
  • For Word and Outlook focus will be moved to the Text Window once the WinBlit Window is opened from those applications.
  • In Internet Explorer you can move focus to the WinBlit Window by pressing the Shift and Tab keys simultaneously until the cross at the top left of the Window appears sunken. Click the tab key one more time and the WinBlit Window will be in focus allowing you to use accelerator keys.
