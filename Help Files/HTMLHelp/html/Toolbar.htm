Toolbar

The Toolbar appeats at the top of the speedreading window and contains on it a number of icons - each of which performs a different function when clicked with the mouse.

Start Button

Click this button to start reading.

Pause Button

Click this button to pause reading. If reading is already paused then clicking this button will restart.

Stop Button

Click this button to stop reading. Click it again to return to the top of the document that you are reading,

Speed Button

Click this button to adjust current speed of reading.

Settings Button

Click this button to change the settings of the current configuration of WinBlit.

Change Configuration Button

Click this and you will be presented with a drop-down menu of all of the current configurations in the system. Select the one you would like to change to from this drop-down menu.

Help Button

Click this button to obtain help for how to use WinBlit.

Register Button

Click this button to register your copy of WinBlit and create a fully licensed version.
