BlitPad

BlitPad is a standalone application which allows you to speed read text files or the contents of the Windows clipboard. To start BlitPad select the ‘Start Menu’ then ‘Program Files’ then ‘WinBlit’ then ‘BlitPad’. Select the File Menu then ‘Open’ to select a file you would like to read. Alternatively select Paste to paste into BlitPad text which you have already copied in from another data source.

If you would like to read more than one document select the ‘New Window’ option from the File Menu. This will open up another copy of BlitPad.

If you would like to read some text which you have selected in the clipboard select the Paste option from the Edit menu.
