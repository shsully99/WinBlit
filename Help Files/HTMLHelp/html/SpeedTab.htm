Speed Tab

This tab allows you to change the speed at which Fixations move across the screen in the Text Window.

Speed

Speed is measured in fixations per minute. The time of one fixation is the time the system takes to highlight then blank out a fixation. The greater the number of fixations per minute the faster you are reading. You can change its value by either moving the slider bar or entering a value in the edit box.

Fixation time Details

Fixation time details describe the proportion of cycle time spent highlighting the text in a fixation. You can change its value either by using the Highlight trackbar or by entering an absolute value of milliseconds in the ‘Highlight Time’ edit box.
