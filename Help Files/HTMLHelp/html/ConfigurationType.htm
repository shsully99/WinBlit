Configuration Type

A Configuration Type defines how fixations are displayed to a user.

Each Configuration that you select has an associated Configuration Type. There are 3 possible values for Configuration Type.

You can see the Configuration Type for the current Configuration by opening the Settings property sheet and looking at the Configuration Type.

RSVP

Fixations stay in the centre of the screen.Text from the document is flashed to you by WinBlit rather than in traditional reading where your eyes move to the text.  The whole page of text is not on view for RSVP Configuration Types though the user can see the progression through the page by selecting the Show Metafile option on the context menu.

Tachistoscopic

This method is only recommended for those who are already familiar with speed reading techniques.

Fixations move from left to right across the Text Window and then drop down to the left to start again on the next section of text. This movement is the same as a normal eye movement for a traditional reader.

Slowhand

This method is only recommended for those who are already familiar with speed reading techniques .

Fixations alternate movement between left to right and right to left across the Text Window dropping down to the next row of text once a traversal is complete.

This is also sometimes known as Lazy-S or S-Shaped.
