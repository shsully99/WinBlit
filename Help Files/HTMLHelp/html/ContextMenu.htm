Context Menu

Right-mouse click anywhere on the WinBlit Window to bring up the context menu

Start/Stop

Select this to start reading. If WinBlit is already  reading then this will cause it to stop.

Pause

Select this to pause reading. If already paused Select this again to restart.

Speed

Select this to adjust speed of reading.

Show Toolbar

Select this to toggle whether the Toolbar is shown in the window.

Show Trackbar

Select this to toggle whether the Trackbar is shown in the window.

Show Statusbar

Select this to toggle whether the Statusbar is shown in the window.

Show Metafile

Select this to toggle whether the Metafile is shown in the window.

Clear Statistics

Select this to reset the statistics that have been accumulated on the Statusbar.

Settings

Select this to change the settings of the current configuration of WinBlit.

Configuration

Select this and you will be presented with a drop-down menu of all of the current configurations in the system. Select the one you would like to change to from this drop-down menu.

Help

Click this item for Help on how to use WinBlit.

About

Click this to view the About Box.
