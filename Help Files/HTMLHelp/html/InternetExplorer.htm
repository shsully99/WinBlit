Internet Explorer

From Internet Explorer you can create the WinBlit Window inside a CommBand to speedread the web page.

To bring up the WinBlit window from within the browser click the WinBlit toolbar button which you will find at the top of the Browser  - You can also do this from the view menu.

NOTE : If you would like to read just a portion of the text of the word document then you can select it with the mouse (as you usually do when selecting text in a word processor). Start up WinBlit to just bring the selected text into the WinBlit window.

Note: After installation to make the toolbar button visible you may need to use the Customise ToolBar dialog box. To do this select the View Menu then ‘ToolBars’ then ‘Customise’.

CommBand

A CommBand is a window that  appears on the bottom of the Internet Explorer (much like the Internet Explorer ‘Tip of the Day’ option). The WinBlit Window appears with the text of the current web page within it. You can then read the web document text in the WinBlit Text Window.
