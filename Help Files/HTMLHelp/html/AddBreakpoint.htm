Add Breakpoint Dialog

This dialog allows you to add a new Breakpoint to the system.

 

Text

 

Enter the text string you wish reading to pause or delay at. This field is case sensitive.

Type

Select one of the 2 radio buttons. Select ‘Pause at Text’ if you would like the reading to pause when it encounters the text you enter. Select ‘Delay at Text’ if you would like the system to delay for a specified period. If you select ‘Delay at Text’ you must enter the number of milliseconds (thousandths of a second) you wish reading to delay for.
