Introduction

WinBlit Speedread is an application that allows you to read Microsoft Office and Internet documents more comfortably and quickly. At the heart of the application is the WinBlit Text Engine which uses a speed reading technique known as RSVP (Rapid Serial Visual Presentation) to display text to you in a manner which can be customised infinitely to suit your personal reading requirements.  With RSVP text from the document comes to your eyes rather than your eyes going to the text.

WinBlit Speedread is supported in Microsoft Word, Outlook, Internet Explorer and as a standalone application called BlitPad which handles several file formats.
